# 🚀 Aula de Arquitetura de Software com SSR (Next.js)

Este projeto é uma aplicação construída com Next.js utilizando o conceito de **Server-Side Rendering (SSR)**. Foi criado com base no `create-next-app` e serve como material prático para estudo de arquitetura moderna de aplicações web.

---

## 📦 Tecnologias Utilizadas

* Next.js
* React
* Node.js
* TypeScript

---

## ⚙️ Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/douglasrohden/aulaArquiteturaSoftwareSSR.git
```

Acesse a pasta do projeto:

```bash
cd aulaArquiteturaSoftwareSSR
```

Instale as dependências:

```bash
npm install
# ou
yarn install
# ou
pnpm install
# ou
bun install
```

Execute o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra no navegador:

👉 [http://localhost:3000](http://localhost:3000)

---

## 🧠 Estrutura do Projeto

* `app/page.tsx` → Página principal da aplicação
* `app/` → Estrutura baseada no App Router (Next.js 13+)
* Atualização automática ao editar os arquivos (Hot Reload)

---

## 🔤 Otimização de Fontes

O projeto utiliza o recurso `next/font` para otimização automática de fontes, incluindo a fonte **Geist**, desenvolvida pela Vercel.
